## Tips

- **Resolution:** 2x retina captures (≥2400px wide) so they render crisp on GitHub.
- **Light mode:** n8n canvas in light mode reads better in screenshots than dark.
- **Crop tight:** strip empty canvas; let the diagram fill the frame.
- **Annotations:** use red boxes / arrows sparingly to highlight a single point per shot.
